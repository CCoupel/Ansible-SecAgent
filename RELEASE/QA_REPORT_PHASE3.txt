✓ API Compatibility
  - /api/exec endpoint (POST, host+command+stdin)
  - /api/upload endpoint (POST, host+dest+data_b64+mode)
  - /api/fetch endpoint (POST, host+src)
  - /api/inventory endpoint (GET, JWT with role "plugin")

✓ Security
  - JWT loaded from file (RELAY_TOKEN_FILE env var)
  - TLS verification configurable (RELAY_CA_BUNDLE)
  - No hardcoded credentials
  - Base64 encoding for file transfers

✓ Error Handling
  - Missing JWT file → raises error
  - File too large → raises error
  - HTTP errors from relay server → propagate
  - Connection failures → raise AnsibleConnectionFailure
